# ChatGPT Backup / Export SEO Plan

## Goal

Strengthen KeepChat AI around the search cluster for ChatGPT conversation backup, chat history export, local archive, and searchable records.

## Target Search Clusters

### Core Commercial

- ChatGPT 聊天记录备份
- ChatGPT 会话备份
- ChatGPT 自动备份
- ChatGPT 本地备份
- ChatGPT 历史记录备份

### Export Intent

- ChatGPT 聊天记录导出
- ChatGPT 对话导出
- ChatGPT 导出 Markdown
- ChatGPT 导出 JSON
- ChatGPT 批量导出

### Archive / Management Intent

- ChatGPT 会话归档
- ChatGPT 历史记录管理
- ChatGPT 聊天记录搜索
- ChatGPT 本地归档

### Comparison / Guide Intent

- ChatGPT 官方导出
- ChatGPT 导出数据
- ChatGPT 官方导出 vs 实时备份

## Page Map

| URL | Primary Keyword | Role |
| --- | --- | --- |
| `/chatgpt-backup` | ChatGPT 聊天记录备份 | Core product landing page |
| `/chatgpt-export` | ChatGPT 聊天记录导出 | Export format and workflow landing page |
| `/chatgpt-archive` | ChatGPT 会话归档 | Archive, search, and long-term management landing page |
| `/guides/chatgpt-official-export-vs-keepchat` | ChatGPT 官方导出 | Comparison guide and education page |

## On-Page Rules

- Use the primary keyword in the title, meta description, H1, first paragraph, one H2, and FAQ copy.
- Rotate related terms naturally: 聊天记录, 会话, 对话, 历史记录, 归档, 导出, 备份, 本地保存.
- Avoid keyword stuffing. The page should read like a clear product guide, not a keyword list.
- Keep each landing page focused on one user intent.

## Internal Link Rules

- Homepage navigation links to backup, export, archive, degraded, and install.
- Homepage feature cards link to the closest SEO page.
- Each SEO landing page links laterally to the other pages in the cluster.
- Bottom related links should reinforce the topic cluster:
  - Backup -> Export, Archive, Privacy, Install
  - Export -> Backup, Archive, Official export guide, Install
  - Archive -> Backup, Export, Library, Privacy
  - Official export guide -> Export, Backup, Archive
